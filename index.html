Over to you

Step 1

Pick something you care about

A side project, a cause, a hobby, a fake business — anything. You need one sentence describing what it is.

Open your AI tool of choice: ChatGPT, Claude, Gemini — whatever you have access to.

Step 2

Give it the vaguest possible brief

Copy this prompt and replace [your thing] with your one sentence:

Build me a simple landing page for [your thing]. Give me the full HTML in a single file that I can open in my browser.
Tap to copy

Mac

  1. Copy all the code it gave you
  2. Open TextEdit → Format → Make Plain Text
  3. Paste the code in
  4. File → Save. Name it mysite.html
  5. Double-click the file to open in your browser

Windows

  1. Copy all the code it gave you
  2. Open Notepad
  3. Paste the code in
  4. File → Save As. Name it mysite.html (change "Save as type" to All Files)
  5. Double-click the file to open in your browser
Step 3

Tell it what you like

  • Share references. Show it what you like. A website, a screenshot, a vibe.
  • Give specific feedback. Not "I don't like it" but "the header feels too corporate, I want something warmer."
  • Assign it a role. Make it your lead designer, not a random assistant.
I need you to act as my lead designer.

I'm building a landing page for [your thing]. Here's what I want it to feel like: [describe 2-3 websites or aesthetics you like].

The current version feels too generic. I want it to feel [your adjectives — e.g. warm, bold, playful, minimal].

Rebuild the full HTML in a single file.
Tap to copy
Step 4

Find your fonts

Ask your AI to suggest 3 font pairings for your site — a headline font and a body font. Tell it the vibe you're going for.

Suggest 3 Google Font pairings for my site. For each one, give me the headline font and the body font, and a one-line description of the vibe.

I want something that feels [your adjectives].
Tap to copy
Step 5

See them side by side

Don't try to decide from a list. Ask it to show you.

Build me a single HTML page that shows all 3 font pairings side by side. Use my site's headline and a paragraph of body copy so I can compare them properly.
Tap to copy
Step 6

Your brand DNA

Your website's HTML already contains your fonts, your colours, your layout. That's not just a website — it's a design system. Point your AI at it and it can make anything in the same style.

Here's the HTML for my website:

[paste your HTML]

Using the same fonts, colours and style, create a simple square social media graphic that says [what you want it to say]. Give me the full HTML in a single file.
